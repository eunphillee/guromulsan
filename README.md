# 구로물산 홈페이지

디지털임베디드 개발 업체 구로물산의 공식 홈페이지입니다.

## 기술 스택

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS (예정)

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 프로젝트 구조

```
guromulsan/
├── app/              # Next.js App Router
├── components/       # React 컴포넌트
├── public/          # 정적 파일
└── styles/          # 전역 스타일
```
