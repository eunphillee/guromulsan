# 배경 음악 파일 안내

이 폴더에 배경 음악 파일을 추가해주세요.

## 파일 형식
- `background-music.mp3` (권장)
- `background-music.ogg` (대체 형식)

## 음악 추천 사이트 (무료)
1. **Pixabay Music** - https://pixabay.com/music/
   - 검색어: "ambient", "technology", "electronic", "calm"
   - 상업적 사용 가능, 저작권 없음

2. **Free Music Archive** - https://freemusicarchive.org/
   - 검색어: "ambient", "electronic", "background"
   - 라이선스 확인 필요

3. **YouTube Audio Library** - https://www.youtube.com/audiolibrary
   - 무료 음악 다운로드 가능
   - 검색어: "ambient", "electronic", "calm"

4. **Incompetech** - https://incompetech.com/music/
   - Kevin MacLeod의 무료 음악
   - 검색어: "ambient", "electronic"

## 음악 선택 가이드
- **장르**: Ambient, Electronic, Chill, Background Music
- **템포**: 느리고 잔잔한 음악 (60-80 BPM)
- **길이**: 2-5분 (루프 재생)
- **분위기**: 기술적이고 미래지향적인 느낌

## 파일 추가 방법
1. 위 사이트에서 적합한 음악 다운로드
2. 파일명을 `background-music.mp3`로 변경
3. 이 폴더(`public/audio/`)에 저장

## 참고
- 음악 파일이 없어도 사이트는 정상 작동합니다
- 음악은 Hero 섹션(첫 페이지)에서만 재생됩니다
- 스크롤하여 Hero 섹션을 벗어나면 자동으로 멈춥니다
