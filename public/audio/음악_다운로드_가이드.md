# 배경 음악 다운로드 가이드

## 빠른 다운로드 방법

### 방법 1: Pixabay Music (가장 추천)
1. https://pixabay.com/music/ 접속
2. 검색창에 다음 키워드 입력:
   - "ambient technology"
   - "electronic background"
   - "calm tech"
   - "corporate ambient"
3. 원하는 음악 선택
4. "Free Download" 클릭
5. 다운로드한 파일을 `background-music.mp3`로 이름 변경
6. `public/audio/` 폴더에 저장

### 방법 2: Orange Free Sounds
1. https://orangefreesounds.com/background-electronic-modern-music/ 접속
2. "Download" 버튼 클릭
3. 파일명을 `background-music.mp3`로 변경
4. `public/audio/` 폴더에 저장

### 방법 3: Dredstudio
1. https://dredstudio.com/tech-corporate-ambient-background/ 접속
2. "Download" 버튼 클릭
3. 파일명을 `background-music.mp3`로 변경
4. `public/audio/` 폴더에 저장

## 파일 저장 위치
```
public/
  └── audio/
      └── background-music.mp3  ← 여기에 저장!
```

## 확인 방법
1. 파일을 저장한 후 개발 서버 재시작
2. 브라우저에서 페이지 로드
3. 브라우저 개발자 도구(F12) → Console 탭 확인
4. "배경 음악 재생 시작" 메시지가 보이면 성공!

## 주의사항
- 파일명은 정확히 `background-music.mp3`여야 합니다
- 파일 형식은 MP3 또는 OGG를 권장합니다
- 음악 파일이 없어도 사이트는 정상 작동합니다
